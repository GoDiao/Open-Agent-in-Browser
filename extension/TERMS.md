> 最后更新：2026-06-04

## 服务描述

Iris 是一个 Chrome 浏览器扩展，提供 AI 辅助的浏览器自动化功能。通过自然语言交互，Iris 可以帮助您：

- 浏览网页并提取信息
- 管理书签、历史记录
- 执行自动化任务
- 记住您的偏好和习惯

## 使用要求

- 您必须拥有兼容的 Chrome 浏览器（Chrome 88+）
- 您需要自行配置有效的 LLM API Key 才能使用 AI 功能
- 您必须年满 13 岁才能使用本软件
