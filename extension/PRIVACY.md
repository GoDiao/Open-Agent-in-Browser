# 隐私政策

> 最后更新：2026-06-04

## 概述

Iris 是一个运行在 Chrome 浏览器中的 AI 助手扩展。本隐私政策说明了我们如何收集、使用、存储和保护您的数据。

## 数据收集

### 您提供的信息

- **用户画像信息**：姓名、昵称、邮箱、时区、语言、职业（通过对话或设置主动提供）
- **AI 性格偏好**：您对 Iris 沟通方式、行为边界的偏好设置
- **配置信息**：API Key、模型选择、主题设置

### 自动收集的信息

- **对话历史**：您与 Iris 的每次对话记录
- **记忆条目**：Iris 从对话中提取并保存的信息
- **选中文本**：您在浏览器中选中的文本（用于智能辅助）
- **使用数据**：工具调用记录、执行历史（本地存储）

### 通过工具收集的信息

Iris 可以执行浏览器操作以帮助您完成任务，包括：
- 浏览历史记录
- 书签
- Cookies 和存储数据
- 当前页面内容

这些数据仅在您明确请求或授权时才会被访问。

## 数据存储

- **存储位置**：所有数据存储在您浏览器的 `chrome.storage.local` 中
- **存储方式**：明文存储（���经加密）
- **数据保留**：除非您手动删除，否则数据将永久保留

## 数据共享

### 第三方 LLM API

当您使用 Iris 的 AI 功能时，对话内容会被发送至您配置的 LLM API 提供商（如 OpenAI、Anthropic、Ollama 等）。

- 我们不收集、上传任何数据至自有服务器
- 您的对话内容仅用于生成 AI 响应
- 请查阅您所使用的 LLM 提供商的隐私政策

### 本地处理

所有数据处理均在您的本地浏览器中完成，我们不会将您的个人数据分享给任何第三方广告商或数据分析公司。

## 用户权利

### 访问权

您可以通过以下方式查看所有存储的数据：
- Memory 面板：查看用户画像和记忆条目
- Settings：查看配置信息
- 对话历史：查看所有会话记录

### 删除权

您可以：
- 在 Memory 面板中删除单个条目
- 在各面板中删除对应数据
- 卸载扩展：清除所有存储数据

### 导出权

（即将推出）您将能够导出所有存储的数据为 JSON 格式。

### 选择退出

- **自动记忆提取**：可在 Settings 中关闭
- **选中文本捕获**：可在 Settings 中禁用

## 数据安全

- 数据存储在您设备的浏览器中，受您设备的整体安全保护
- 建议您：
  - 保护好您的设备访问权限
  - 不要在共享设备上使用 Iris
  - 定期清理不需要的记忆条目

## 儿童隐私

Iris 不面向 13 岁以下儿童，我们也不会故意收集儿童的个人信息。

## 隐私政策变更

如果我们对本隐私政策进行重大变更，我们将通过以下方式通知您：
- 在扩展设置中发布通知
- 在更新日志中说明

## 联系我们

如果您对本隐私政策有任何疑问，请通过 GitHub Issues 与我们联系。
